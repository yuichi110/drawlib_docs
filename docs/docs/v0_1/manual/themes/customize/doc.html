Customize official theme

You can create your own theme from scratch. But if you don’t need huge modification from official themes, customization might be good.

Update style objects

You can access to theme style objects through dtheme.<type>styles. Where the <type> is equivalent to Style class name. For example, IconStyle can be accessed with dtheme.iconstyles.

Each objects posses these methods.

  • has(name): whether the name style exists or not

  • list(): get names of styles

  • get(name): get style object

  • set(name): set style object

  • delete(name): delete style object

Here is a update procedure.

  1. Get original object. Such as textstyle = dtheme.textstyles.get(name)

  2. Modify style. Such as textstyle.size = 24

  3. Repeat modification

  4. Set modified object. Such as dtheme.textstyles.set(textstyle, name)

Let’s take a look simple example which change text style.

image1_1.py
 1from drawlib.apis import *
 2
 3config(width=100, height=50)
 4
 5# before change
 6text((25, 15), "Hello Drawlib!!")
 7text((25, 35), "Hello Drawlib!!", style="green")
 8
 9# change theme style
10default_text_style = dtheme.textstyles.get()
11default_text_style.size = 24
12default_text_style.font = FontSansSerif.RALEWAYS_REGULAR
13dtheme.textstyles.set(default_text_style)
14green_text_style = dtheme.textstyles.get("green")
15green_text_style.size = 24
16green_text_style.font = FontSansSerif.RALEWAYS_REGULAR
17dtheme.textstyles.set(green_text_style, "green")
18
19# after change
20text((75, 15), "Hello Drawlib!!")
21text((75, 35), "Hello Drawlib!!", style="green")
22
23save()

You can update style one by one. However, using dtheme.<type>styles.list() and for loop is useful for changing all styles with same update.

image1_2.py
 1from drawlib.apis import *
 2
 3config(width=100, height=50)
 4
 5# before change
 6text((25, 15), "Hello Drawlib!!")
 7text((25, 35), "Hello Drawlib!!", style="green")
 8
 9# change all theme text styles
10for name in dtheme.textstyles.list():
11    text_style = dtheme.textstyles.get(name)
12    text_style.size = 24
13    text_style.font = FontSansSerif.RALEWAYS_REGULAR
14    dtheme.textstyles.set(text_style, name)
15
16# after change
17text((75, 15), "Hello Drawlib!!")
18text((75, 35), "Hello Drawlib!!", style="green")
19
20save()
