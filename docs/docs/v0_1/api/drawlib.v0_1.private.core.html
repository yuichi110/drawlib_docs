drawlib.v0_1.private.core package

drawlibs core package contains core code of drawing.

drawlib use matplotlib for drawing. core package contains its related codes.

Submodules

drawlib.v0_1.private.core.colors module

Color definition module

drawlib only supports these 2 style of colors.

  • RGB (R, G, B) : Each channel supports 0 ~ 255.

  • RGBA (R, G, B, A) : Each channel supports 0 ~ 255. Alpha supports 0.0 ~ 1.0.

Technically library can support string color name(red, blue etc) and hexcode(#00FF00 etc). However, supporting many format makes messhy implementation. So, providing Color class and let them use supported style without stress.

This module posses 1 base class and 2 Color classes

  • BaseColors: base class of Colors. User can inherit it.

  • Colors: 16 Basic web colors

  • Colors140: 140 web colors

User can define his color class if they need. Please dont use Enum for colors. Just RGB with tuple(int, int int) is good.
