Drawing lineÔÉÅ

Drawlib has 5 line functions.

  • line()

  • line_curve()

  • line_bezier1()

  • line_bezier2()

  • lines()

  • lines_bezier()

We will explain them one by one. All of them takes these style classes optionally.

  • LineStyle

  • LineArrowStyle

As these name explains, LineStyle is for line and LineArrowStyle is for arrow line. If style is not provided, default line style is applied. We explain these styles on other page.

line()ÔÉÅ

Function line() is a most basic function which draw line. It takes 3 arguments.

  • xy1: start point

  • xy2: end point

  • style

Lets‚Äôs check by example.

image1.pyÔÉÅ
1from drawlib.apis import *
2
3config(width=100, height=50)
4line(xy1=(10, 10), xy2=(90, 40))
5save()

We draw line from (10, 10) to (90, 40) without specifying style. It generate this output.

line_curve()ÔÉÅ

Function line_curve() is easy to draw curved line. Functions line_bezier1() and line_bezier2() are also able draw curved lines. However they requires complex curve controlling rather than this function.

This function takes 4 arguments.

  • xy1

  • xy2

  • bend: additional length rather than direct connection

  • style

For example, suppose xy1 is (10, 10) and xy2 is (10, 20). The distance of them is 10. When we set bend 0.2, draw line from (10, 10) to (10, 20) with length 12(10 x 1.2). When we set bend 0.4, length is 14(10 x 1.4).

We can also set minus value to bend. The length is same to plus value, but bending side becomes reverse. Let‚Äôs check examples.

image2.pyÔÉÅ
 1from drawlib.apis import *
 2
 3config(width=100, height=50)
 4line_curved(xy1=(10, 20), xy2=(90, 20), bend=0.4)
 5text((50, 7), "0.4")
 6line_curved(xy1=(10, 23), xy2=(90, 23), bend=0.2)
 7text((50, 18), "0.2")
 8line_curved(xy1=(10, 27), xy2=(90, 27), bend=-0.2)
 9text((50, 32), "-0.2")
10line_curved(xy1=(10, 30), xy2=(90, 30), bend=-0.4)
11text((50, 43), "-0.4")
12save()

lines_bezier()ÔÉÅ

Function lines_bezier() is similar to lines(). But it can draw bezier lines from points to points. It takes these arguments.

  • xy: start point

  • path_points: list of ‚Äútuple of bezier line‚Äù

  • style

Difficult part is path_points. It takes 3 type of tuple.

  • (x, y): draw last point to (x, y)

  • (cp_x, cp_y, x, y): draw bezier1 line from last point to (x, y) with 1 control point

  • (cp1_x, cp1_y, cp2_x, cp2_y, x, y): draw bezier2 line from last point to (x, y) with 2 control points

Let‚Äôs check the how it works with example.

image6.pyÔÉÅ
 1from drawlib.apis import *
 2
 3config(width=100, height=50)
 4
 5points = [
 6    (10, 20, 30, 20),
 7    (60, 10),
 8    (60, 40, 90, 40, 90, 10),
 9]
10lines_bezier(xy=(10, 40), path_points=points)
11
12# bezier1 help line
13line(xy1=(10, 40), xy2=(10, 20), style=LineStyle(style="dashed", color=Colors.Red))
14line(xy1=(10, 20), xy2=(30, 20), style=LineStyle(style="dashed", color=Colors.Red))
15circle(xy=(10, 40), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
16circle(xy=(10, 20), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
17circle(xy=(30, 20), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
18
19# bezier2 help line
20line(xy1=(60, 10), xy2=(60, 40), style=LineStyle(style="dashed", color=Colors.Red))
21line(xy1=(60, 40), xy2=(90, 40), style=LineStyle(style="dashed", color=Colors.Red))
22line(xy1=(90, 40), xy2=(90, 10), style=LineStyle(style="dashed", color=Colors.Red))
23circle(xy=(60, 10), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
24circle(xy=(60, 40), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
25circle(xy=(90, 40), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
26circle(xy=(90, 10), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))

We use 3 type of tuple as element of path_points. They becomes line, bezier1 line, bezier2 line. Executing this code generate this output.

This function might be useful when you want to draw curved line from shape to shape.

image7.pyÔÉÅ
 1from drawlib.apis import *
 2
 3config(width=100, height=50)
 4
 5circle((10, 40), radius=5)
 6lines_bezier(
 7    (20, 40),
 8    path_points=[
 9        (75, 40),
10        (90, 40, 90, 25),
11        (90, 20),
12    ],
13    style=LineArrowStyle(hstyle="-|>"),
14)
15circle((90, 10), radius=5)
16
17# bezier1 help line
18line(xy1=(75, 40), xy2=(90, 40), style=LineStyle(style="dashed", color=Colors.Red))
19line(xy1=(90, 40), xy2=(90, 25), style=LineStyle(style="dashed", color=Colors.Red))
20circle(xy=(75, 40), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
21circle(xy=(90, 40), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))
22circle(xy=(90, 25), radius=0.5, style=ShapeStyle(fcolor=Colors.White, lcolor=Colors.Red))

24save()

We use 3 type of tuple as element of path_points. They becomes line, bezier1 line, bezier2 line. Executing this code generate this output.

If you requires precise control, please use this function. However, if you want to draw simple curve line, we recommend using lines_curved() instead.
