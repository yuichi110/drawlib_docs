Line and Arrow Style

All functions which draw line takes style option as argument. We supports these 2 style class.

  • LineStyle

  • LineArrowStyle

As there name shows, LineStyle is used for drawing line. And LineArowStyle is used for drawing line which has arrow head.

We have function arrow() too. But it is thick and bold arrow shape. Not an arrow line.

LineStyle

LineStyle has these attributes.

  • width: Line width. float value

  • color: Line color. RGB (0~255, 0~255, 0~255) or RGBA (0~255, 0~255, 0~255, 0~1.0). You can use Color class

  • alpha: Line alpha. 0.0 ~ 1.0. 0 is totally transparent

  • style: Line style, such as solid, dash etc

Let’s check by examples.

image1.py
 1from drawlib.apis import *
 2
 3config(width=100, height=40)
 4
 5text((10, 5), "no style")
 6line((20, 5), (40, 5))
 7text((10, 15), "width: 5")
 8line((20, 15), (40, 15), style=LineStyle(width=5))
 9text((10, 25), "color: Red")
10line((20, 25), (40, 25), style=LineStyle(color=Colors.Red))
11text((10, 35), "alpha: 0.2")
12line((20, 35), (40, 35), style=LineStyle(alpha=0.2))
13
14text((60, 5), "style: solid\n(default)")
15line((70, 5), (90, 5), style=LineStyle(style="solid"))
16text((60, 15), "style: dashed")
17line((70, 15), (90, 15), style=LineStyle(style="dashed"))
18text((60, 25), "style: dotted")
19line((70, 25), (90, 25), style=LineStyle(style="dotted"))
20text((60, 35), "style: dashdot")
21line((70, 35), (90, 35), style=LineStyle(style="dashdot"))
22
23save()

We change style 1 by 1. But you can change style at same time such as LineStyle(color=Colors.Red, width=5, style="dashed").

You can see line styles

LineArrowStyle

LineArrowStyle is almost similar to LineStyle. But it has few additional attributed and having change of original attribute. It has these attributes.

  • lwidth: Line width

  • lstyle: Line style

  • hstyle: Arrow head style

  • hscale: Arrow head scale (size)

  • color: Arrow line color

  • alpha: Arrow line alpha

To distinguish whether line and arrow head parameter, few attribute starts from l(line) and others starts from h(head). hstyle can have these values.

  • "->:: Arrow at end point.

  • "<-:: Arrow at start point.

  • "<->:: Arrow at both start and end point.

  • "-|>:: Filled arrow at end point.

  • "<|-:: Filled arrow at start point.

  • "<|-|>:: Filled arrow at both start and end point.

Let’s check by example.

image2.py
 1from drawlib.apis import *
 2
 3"""
 4config(width=100, height=50)
 5
 6text((10, 5), "LineArrowStyle()", style=TextStyle(size=14))
 7line((20, 5), (40, 5), )
 8text((10, 13), "width: 5")
 9line((20, 13), (40, 13), style=LineArrowStyle(lwidth=5))
10text((10, 21), "color: Red")
11line((20, 21), (40, 21), style=LineArrowStyle(color=Colors.Red))
12text((10, 29), "alpha: 0.2")
13line((20, 29), (40, 29), style=LineArrowStyle(alpha=0.2))
14text((10, 37), "lstyle: dashed")
15line((20, 37), (40, 37), style=LineArrowStyle(lstyle="dashed"))
16text((10, 45), "hscale: 50\n(default: 20)")
17line((20, 45), (40, 45), style=LineArrowStyle(hscale=50))
18
19text((60, 5), "hstyle: ->\n(default)")
20line((70, 5), (90, 5), style=LineArrowStyle(hstyle="->"))
21text((60, 13), "style: <-")
22line((70, 13), (90, 13), style=LineArrowStyle(hstyle="<-"))
23text((60, 21), "style: <->")
24line((70, 21), (90, 21), style=LineArrowStyle(hstyle="<->"))
25text((60, 29), "style: -|>")
26line((70, 29), (90, 29), style=LineArrowStyle(hstyle="-|>"))
27text((60, 37), "style: <|-")
28line((70, 37), (90, 37), style=LineArrowStyle(hstyle="<|-"))
29text((60, 45), "style: <|-|>")
30line((70, 45), (90, 45), style=LineArrowStyle(hstyle="<|-|>"))
31
32save()
33"""

We change style 1 by 1. But you can change style at same time such as LineArrowStyle(color=Colors.Red, lwidth=5, lstyle="dashed", hstyle="<->").

To draw default arrow line, please pass LineArrowStyle() to line. If you don’t specify style, normal line will be drawn.
