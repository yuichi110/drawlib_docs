Define your own theme

We provides official themes. But some users may want to create your own theme. You can achieve it with these 2 ways.

  • Register your style objects manually

  • Syntax sugar of above

We recommend using syntax sugar way at styling code which is explained at “building many images” doc.

dtheme.apply_custom_theme

dtheme.apply_custom_theme() provides setting custom theme feature. It takes these arguments.

  • default_style:dtheme.ThemeStyle

  • named_styles:List[Tuple[str, dtheme.ThemeStyle]]

  • theme_colors:List[Tuple[str, Tuple[int, int, int]]]

The type dtheme.ThemeStyle is data class which holds all drawlib styles. All member of the class is optional (default is None). But data which is passed to arg default_style requires them.

  • backgroundcolor

  • sourcecodefont

  • iconstyle

  • imagestyle

  • linestyle

  • linearrowstyle

  • shapestyle

  • shapetextstyle

  • textstyle

You can get these styles with dtheme.<styles>.get(). Those value becomes default.

On the other hand, named_styles is a list of tuple. The tuple posses name and dtheme.ThemeStyle object. You can get those styles via dtheme.<styles>.get(name).

OK, let’s check example custom theme which uses gold and silver colors.

image1.py
 1from drawlib.apis import *
 2
 3default = dtheme.ThemeStyle(
 4    backgroundcolor=Colors140.AliceBlue,
 5    sourcecodefont=FontSourceCode.ROBOTO_MONO,
 6    iconstyle=IconStyle(style="fill", color=Colors140.Gold),
 7    imagestyle=ImageStyle(lcolor=Colors140.Gold, lwidth=0),
 8    linestyle=LineStyle(width=3, color=Colors140.Gold),
 9    linearrowstyle=LineArrowStyle(lwidth=3, color=Colors140.Gold),
10    shapestyle=ShapeStyle(lwidth=0, lcolor=Colors140.Black, fcolor=Colors140.Gold),
11    shapetextstyle=ShapeTextStyle(color=Colors140.DarkRed, font=FontSansSerif.RALEWAYS_BOLD),
12    textstyle=TextStyle(color=Colors140.DarkRed, font=FontSansSerif.RALEWAYS_BOLD),
13)
14
15gold = dtheme.ThemeStyle(
16    iconstyle=IconStyle(style="fill", color=Colors140.Gold),
17    imagestyle=ImageStyle(lcolor=Colors140.Gold, lwidth=0),
18    linestyle=LineStyle(width=3, color=Colors140.Gold),
19    linearrowstyle=LineArrowStyle(lwidth=3, color=Colors140.Gold),
20    shapestyle=ShapeStyle(lwidth=0, lcolor=Colors140.Black, fcolor=Colors140.Gold),
21    shapetextstyle=ShapeTextStyle(color=Colors140.Gold, font=FontSansSerif.RALEWAYS_BOLD),
22    textstyle=TextStyle(color=Colors140.Gold, font=FontSansSerif.RALEWAYS_BOLD),
23)
24
25silver = dtheme.ThemeStyle(
26    iconstyle=IconStyle(style="fill", color=Colors140.Silver),
27    imagestyle=ImageStyle(lcolor=Colors140.Silver, lwidth=0),
28    linestyle=LineStyle(width=3, color=Colors140.Silver),
29    linearrowstyle=LineArrowStyle(lwidth=3, color=Colors140.Silver),
30    shapestyle=ShapeStyle(lwidth=0, lcolor=Colors140.Black, fcolor=Colors140.Silver),
31    shapetextstyle=ShapeTextStyle(color=Colors140.Silver, font=FontSansSerif.RALEWAYS_BOLD),
32    textstyle=TextStyle(color=Colors140.Silver, font=FontSansSerif.RALEWAYS_BOLD),
33)
34
35dtheme.apply_custom_theme(
36    default_style=default,
37    named_styles=[
38        ("gold", gold),
39        ("silver", silver),
40    ],
41    theme_colors=[
42        ("gold", Colors140.Gold),
43        ("silver", Colors140.Silver),
44    ],
45)
46
47config(width=100, height=50)
48x1 = 20
49x2 = 50
50x3 = 80
51y1 = 10
52y2 = 25
53y3 = 40
54
55line((x1 - 10, y1), (x1 + 10, y1))
56circle((x1, y2), radius=10)
57text((x1, y3), "Hello Drawlib!")
58
59for x, name in [(x2, "gold"), (x3, "silver")]:
60    line((x - 10, y1), (x + 10, y1), style=name)
61    circle((x, y2), radius=10, style=name)
62    text((x, y3), "Hello Drawlib!", style=name)
63
64save()
65clear(initialize_theme=True)

We define theme styles first. After that, calling dtheme.apply_custom_theme().

You can see your theme is applied.
