Official Theme: essentials

Theme essentials has basic 25 colors. It is difficult to remember all colors since theme volume is large. So, we recommend to use this theme for base of your custom theme. You posses lots of option for color and styles.

The style rule is almost same to simple theme default2. If you don’t know the theme, please check it first.

Colors

Theme monochrome posses lots of colors. Color came from Flat UI Colors (https://flatuicolors.com).

  • green_sea
